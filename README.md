# advisee.ai Landing Page

This is a simple landing page for advisee.ai, an AI-powered career guidance platform.

## Features

- Responsive design that works on desktop and mobile devices
- Clean, modern UI with a professional look
- Links to Substack newsletter and Meetup group

## Local Development

To run this site locally:

1. Clone this repository
2. Open the `index.html` file in your browser

## Deployment

This site can be deployed to Vercel by following the instructions in the `vercel-deployment-guide.md` file.
